Steps to test the referral system:

1. Register user
2. Login user-1 { leave referral code blank }
3. Login user-2 { use referral code from user-1 }
4. take subscription for user-2
5. check wallet of user-1

You will find end points in routes folder. Please use postman to test the end points.

-> for subscription end point use jwt token. You can get the token from login end point for that user.
